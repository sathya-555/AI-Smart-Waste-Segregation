# AI Smart Waste Segregation & Sustainability Advisor

## Overview

This project is developed as part of the 1M1B AI for Sustainability Virtual Internship.

The application helps users classify waste items, understand environmental impact, and receive disposal recommendations.

## SDG Alignment

SDG 12 - Responsible Consumption and Production

## Features

- Waste Classification
- Sustainability Score
- Environmental Impact Analysis
- Disposal Guidance
- Image Upload Interface
- Streamlit Web Application

## Technologies Used

- Python
- Streamlit
- IBM BOB
- Generative AI Concepts
- Prompt Engineering

## Run Locally

pip install -r requirements.txt

streamlit run app.py
